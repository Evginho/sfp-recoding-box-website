Up and Running in 3 Steps

1

Assemble & Connect

Mount the SFP cage, connect USB-C to your laptop. Green LED confirms power.

2

Install CLI

# Linux/macOS
pip install sfp-recoding-box

# Windows
winget install networkser.sfp-recoding-box
3

Read Your First Module

sfp --version
sfp detect
sfp read 1 -o backup.bin
sfp info 1
